Getting Started
===============

Python version support
----------------------

Only Python 3.6 and 3.7.

Installing Perfana
------------------

:code:`Perfana` can be installed via pip from `PyPI <https://pypi.org/project/perfana/>`_

.. code-block:: bash

    pip install perfana

Alternatively, you can install it via :code:`conda` with

.. code-block:: bash

    conda install -c danielbok perfana
